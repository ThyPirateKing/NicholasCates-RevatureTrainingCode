As a Dungeon Master (DM) or Player (PC) in Dungeons and Dragons 5th Edition:

I want to print my character sheet or the character sheets of my player(s) other than spells, spell slots, items, notes, and other miscellaneous information.
    - (so most of the first page of a DnD 5th Edition Character Sheet)
So that I can have a physical reference for the character's abilities, skills, and other statistics during gameplay.

Acceptance Criteria:

The application should prompt the user in the console or calculate the value for all relevant character information, including:

-Information Input By User:
    - Name
    - Race
    - Class
    - Background
    - Alignment
    - Level
    - Ability Scores (Strength, Dexterity, Constitution, Intelligence, Wisdom, and Charisma)
    - Armor Class
    - Speed
    - Hit Points, Temporary Hit Points, and Hit Point Maximum (optional)
    - Hit Dice
    - Inspiration
    - Skill Proficiencies
        - (Acrobatics, Animal Handling, Arcana, Athletics, Deception, History, Insight, Intimidation, Investigation,
            Medicine, Nature, Perception, Performance, Persuation, Religion, Sleight of Hand, Stealth, and Survival)

- Calculated Information:
    - Initiative (This is calculated automatically)
    - Proficiency Bonus
    - Ability Score Modifiers
